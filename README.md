Proyecto web que permite a los usuarios subir y gestionar apuntes en una biblioteca digital (LIB).
El acceso a las funcionalidades principales requiere que el usuario esté registrado y autenticado.

Los usuarios registrados pueden subir, descargar, compartir y valorar apuntes. Durante la carga de un apunte, el sistema valida el tamaño y la extensión del archivo, asegurando que solo se permitan formatos autorizados.

Los usuarios no registrados pueden visualizar apuntes en formato PDF mediante un visor integrado, pero no tienen acceso a funciones de descarga, carga o interacción.

El sistema incluye un rol de administrador, encargado de la gestión de usuarios y la eliminación de apuntes cuando sea necesario. Sin embargo, solo el usuario creador de un apunte puede editarlo, incluso el administrador no tiene permisos para modificar contenido ajeno.

Además, los usuarios registrados pueden compartir apuntes con otros usuarios. Quienes reciban un apunte compartido pueden decidir mantenerlo o eliminarlo de su lista de compartidos.
